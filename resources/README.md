# Resources

Curated learning resources for MongoDB, organized by type.

## Official Documentation

- [MongoDB Manual](https://www.mongodb.com/docs/manual/) — The authoritative reference
- [PyMongo Documentation](https://pymongo.readthedocs.io/) — Python driver docs
- [MongoDB Shell (mongosh)](https://www.mongodb.com/docs/mongodb-shell/) — Shell reference
- [Atlas Documentation](https://www.mongodb.com/docs/atlas/) — Cloud platform docs
